# 04 — UX / Interface

## Pages à développer (Stagiaire 1)

### Page Upload — `/appels-offres/nouveau`
- Formulaire glisser-déposer
- Formats acceptés : PDF, DOCX, TXT
- Lancement de l'analyse après soumission
- Redirection vers la page détail

### Page Détail AO — `/appels-offres/{id}`
- Résumé généré par l'IA
- Score de pertinence avec justification
- Compétences requises (tags)
- Bouton téléchargement fichier original
- Bouton Archiver / Marquer comme traité

### Page Notifications — `/notifications`
- Liste des notifications envoyées
- Filtres par type
- Lien vers l'AO concerné
- Possibilité de renvoyer une notification
